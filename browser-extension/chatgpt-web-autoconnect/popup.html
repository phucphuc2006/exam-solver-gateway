NexusAI Web Auto Connect

Kiểm tra extension đã nối được với dashboard Web Bridge cho ChatGPT, Gemini và Grok hay chưa.

Chưa kiểm tra

Mở tab NexusAI `/dashboard/chatgpt-web`, rồi bấm nút bên dưới để kiểm tra kết nối thật.

Popup sẽ dò tab dashboard hợp lệ và ping trực tiếp vào bridge trên trang.
